•	What is LWC? 

Lightning Web Components (LWC) is a modern UI framework by Salesforce used to build fast and efficient user interfaces.

It is based on standard web technologies:

a) HTML 

b) JavaScript (logic)

c) CSS 

LWC follows a component-based architecture, meaning UI is divided into small reusable parts.

-----------------------------------------------------------------------------------------------------------------------------------------------------------------------

•	Why Salesforce Uses LWC 

Salesforce moved to LWC because:

a) High performance (faster than Aura)

b) Uses modern JavaScript (ES6+)

c) Reusable components

d) Better security

e) Easier maintenance and scalability

•	Frontend vs Backend Logic 

**Frontend (UI Layer)**

Button click handling

Displaying data

Basic validation (required fields)

Showing notifications

 **Backend (Apex Layer)**

Database operations (Insert, Update, Delete)

Business logic (fee calculation)

Advanced validation

Security enforcement
